# lvm_multi_api

A library for interacting with multiple Large Vision Model (LVM) APIs.

## Supported APIs

- OpenAI
- XAI
- Automatic1111

## Installation

Add the following to your `Cargo.toml`:

```toml
[dependencies]
lvm_multi_api = "0.1.0"
```

## Usage

Usage examples can be found in the `tests` directory.
